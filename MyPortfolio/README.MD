# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my educational background, technical skills, projects, and professional journey. This portfolio was developed as part of my **Frontend Web Development Internship** at **CodeAlpha** and serves as a digital resume for recruiters, collaborators, and fellow developers.

---

## 📖 Project Overview

This project is a fully responsive personal portfolio website built using **HTML5**, **CSS3**, and **Bootstrap 4**. It reflects both my academic background and my transition into software development.

The portfolio introduces visitors to my professional profile, highlights my technical skills, showcases selected projects, and provides convenient ways to connect with me.

---

## ✨ Features

* 🎨 Modern and clean user interface
* 📱 Fully responsive design for desktop, tablet, and mobile devices
* 🧭 Sticky navigation bar with smooth scrolling
* 🎯 Fixed header offset for section navigation
* 👨‍💻 Professional About Me section
* 🛠️ Technical Skills showcase
* 📂 Interactive Project Portfolio
* 📞 Contact section with social links
* ⚡ Smooth hover animations and transitions
* 🖼️ Optimized responsive images using `object-fit`
* 🌙 High-contrast dual-tone color palette for improved readability

## 💻 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/venkateshmalla1/codealpha_tasks.git
```

### 2. Navigate to the Project

```bash
cd codealpha_tasks/MyPortfolio
```

### 3. Run the Website

#### Option 1: Open Directly

Open the following file in your preferred web browser:

```text
my portipolio.html
```

#### Option 2: VS Code Live Server

1. Open the project in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click on **my portipolio.html**.
4. Select **Open with Live Server**.
