# Personal Portfolio Website — Malla Venkatesh

## 📌 Project Overview
This repository hosts the code for my **Personal Professional Portfolio Webpage**. Designed with a sleek, high-contrast, modern layout, this site serves as an interactive hub highlighting my dual background in humanities and technical engineering[cite: 8, 9]. It outlines my ongoing development journey, project accomplishments, and professional milestones.

---

## ✨ Key Features
* **Dual-Tone Visual Highlighting:** Uses a striking alternate container theme, shifting smoothly from deep slate backgrounds (`#191a1d`) in hero spaces into soft gray palettes (`#f8f9fa`) for readability[cite: 9].
* **Responsive Fluid Navigation:** Fully operational Bootstrap navbar featuring collapse toggles and viewport sticky spacing rules (`fixed-top`)[cite: 8, 9].
* **Jump Link Header Offsets:** Custom implementation of `scroll-padding-top` rules to prevent fixed navigation elements from overlaying section heading contexts[cite: 9].
* **Balanced Aesthetic Typography:** Leverages precise flexbox column wrappers and `object-fit: cover` parameters to yield vertically centered text and responsive grid imagery[cite: 9].
* **Interactive Project Showcases:** Smooth translate transforms (`translateY`) mapped onto portfolio cards to elevate content gracefully upon hover actions[cite: 9].

---

## 🛠️ Technologies & Dependencies Used
* **HTML5:** Semantic architecture laying down layout grids, hyperlinked action tabs, and structured portfolio anchors.
* **CSS3:** Custom stylesheet housing global layout fixes, Google Font family imports (`Roboto`), and smooth element transitions[cite: 9].
* **Bootstrap v4.5.2:** Responsive grid column layouts (`col-12`, `col-md-7`, `order-md-1`) to enable multi-screen responsiveness across mobile, tablet, and widescreen viewports[cite: 8, 9].
* **Cloudinary CDN:** For rapid asset streaming and lightweight delivery of high-resolution graphic assets[cite: 8, 9].
