# 🧮 Basic Calculator

A modern, responsive, and interactive calculator web application built using **HTML5**, **CSS3**, **Bootstrap 4.6**, and **JavaScript (ES6)**. This project was developed as part of my **Frontend Web Development Internship** at **CodeAlpha**. The calculator performs fundamental arithmetic operations with a clean user interface, responsive layout, and robust input validation powered by an Object-Oriented Programming (OOP) approach.

---

# 📖 Project Overview

This project is a fully functional calculator designed to perform basic mathematical computations while delivering a seamless user experience. The application follows modern frontend development practices by separating presentation, styling, and application logic into dedicated files.

The calculator utilizes an object-oriented JavaScript architecture, making the codebase modular, maintainable, and easy to extend with additional mathematical functions in the future.

# ⚙️ Application Workflow

The calculator is built using an Object-Oriented Programming approach where all functionality is encapsulated inside a dedicated `Calculator` class.

### Number Input

* Accepts numeric values from calculator buttons.
* Prevents invalid multiple decimal entries.

### Operation Selection

* Handles arithmetic operators (`+`, `−`, `×`, `÷`).
* Stores previous operand and selected operation.

### Computation Engine

* Performs mathematical calculations.
* Updates the display dynamically.
* Handles division-by-zero scenarios gracefully.

### Display Management

* Maintains previous and current operand displays.
* Formats calculations for improved readability.

### Event Handling

* Registers click events for all calculator buttons.
* Updates the calculator state in real time.
